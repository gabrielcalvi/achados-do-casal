# ROADMAP - ACHADOS DO CASAL

## Objetivo do Projeto

Criar o melhor portal brasileiro de recomendações de produtos, com foco em:
- Mercado Livre
- Amazon
- Shopee
- Magalu
- Demais parceiros afiliados

---

# Metodologia

1. Trabalhar apenas em uma Sprint por vez.
2. Nenhuma funcionalidade nova entra durante a Sprint.
3. Novas ideias vão para o BACKLOG.
4. Toda Sprint termina com:
   - implementação;
   - testes;
   - aprovação;
   - commit.
5. O projeto deve permanecer estável após cada Sprint.
